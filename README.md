# 문장(Paragraph)

원하는 텍스트를 있는 그대로 입력하면 문장이 출력됩니다 별도의 특수문자나 형식이 존재하지 않습니다

<br/>
<br/>

# 줄바꿈(Line Break)

원하는 텍스트를 입력하고 줄바꿈을  
하고 싶은 위치에서 공백을 2칸 입력  
합니다

또한 공백 2칸 입력 대신 `<br>` 태그를 <br/>
사용하여 줄바꿈도 가능합니다

<br/>
<br/>
